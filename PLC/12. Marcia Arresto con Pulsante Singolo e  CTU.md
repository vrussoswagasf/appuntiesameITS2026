# Marcia Arresto con Pulsante Singolo e CTU
---
## Componenti
| Nome      | Tipo di Dato | Indirizzo | Commento |
| --------- | ------------ | --------- | -------- |
| BT        | bool         | %I0.0     |          |
| LED       | bool         | %Q0.0     |          |
| conteggio | int          | %MW2      |          |

## Descrizione
Premendo BT si apre il flusso verso il contatore CTU che con il numero intero 1 associato a PV aprirà il flusso in Q, accendendo poi la bobina in uscita LED dopo una sola pressione del tasto BT. la variabile in uscita da CV, conteggio, conta quante pressioni sono state effettuate su BT, quando il conteggio è maggiore o uguale a 2 il contatore CTU viene resettato a 0, quindi quando premo il pulsante una volta accendo LED se lo premo una seconda volta il contatore si resetta spegnendo LED
