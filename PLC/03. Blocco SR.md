# Blocco SR
---
## Componenti
| Nome     | Tipo di Dato | Indirizzo | Commento |
| -------- | ------------ | --------- | -------- |
| BT1      | bool         | %I0.0     |          |
| BT2      | bool         | %I0.1     |          |
| LEDVERDE | bool         | %Q0.0     |          |
| LEDROSSO | bool         | %Q0.1     |          |
| FFSR     | bool         | %M10.0    |          |
## Descrizione
Il Blocco SR serve a Settare o Resettare una variabile tramite due Input, uno di Set e uno di Reset, in questo caso se premo BT1 setto la bobina LEDVERDE a True, se invece premo il tasto BT2 il Blocco SR viene resettato e il LEDVERDE si spegne.
Sotto c'è un altro segmento dove LEDVERDE è normalmente chiuso ed è collegato alla bobina LEDROSSO, quindi se la bobina LEDVERDE è True il contatto LEDVERDE normalmente chiuso si apre e LEDROSSO resta spento, se invece LEDVERDE va a False essendo la bobina LEDVERDE normalmente chiuso fa passare la "corrente" e LEDROSSO si accede
