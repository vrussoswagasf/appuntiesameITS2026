# Lettura Segnali Analogici 0-10 Volt
---
## Componenti
| Nome      | Tipo di Dato | Indirizzo | Commento |
| --------- | ------------ | --------- | -------- |
| Analogico | Int          | %IW64     |          |
| V1        | Real         | %MD66     |          |
| V2        | Real         | %MD70     |          |
| LED       | bool         | %Q0.0     |          |
## Descrizione
Per leggere segnali analogici nel plc si utilizzano i blocchi NORM_X e SCALE_X.
Il blocco NORM_X si occupa di trasformare un valore analogico in entrata in un numero tra 0 e 1 con virgola tramite la variabile in entrata VALUE e il valore in uscita lo associa alla variabile OUT, il range per l'analogico va da 0 a 32726, successivamente nel blocco SCALE_X si da in ingresso la variabile associata al valore in uscita nel blocco NORM_X, SCALE_X si occuperà di trasformare il numero tra 0 e 1 con virgola in uscita in un numero con intervallo a piacere successivamente dandolo in uscita alla variabile OUT, in questo caso il valore in uscita da SCALE_X sarà un numero compreso tra 0 e 100, nel secondo segmento quando V2 diventerà maggiore o uguale a 50.0 accenderà la bobina in uscita LED
