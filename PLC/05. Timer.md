# Timer
---
## Descrizione
In Ladder esistono 4 tipi di Timer
>[!note] **Timer TP**
>Il Timer TP accende una bobina per tot secondi
>Il timer scatta dopo che riceve un input in ingresso, se l'input in ingresso resta True il timer quando arriva alla fine del tempo prefissato interrompe il flusso alla bobina in uscita

>[!note] **Timer TON**
>Il Timer TON accende una bobina solo se gli viene dato un input in ingresso superiore al tempo stabilito, se poi l'input viene interrotto il Timer TON interrompe il flusso alla bobina in uscita.
>Ad esempio se ho un tempo prefissato di 2sec, premo un tasto e dopo che sono passati due secondi accendo un led, quando smetto di premere il tasto il led si spegnerà

>[!note] **Timer TOF**
>Il Timer TOF accende una bobina per il tempo in cui gli viene dato un input in ingresso più il tempo stabilito, quando l'input in ingresso e il tempo stabilito si interrompono si interromperà il flusso alla bobina in uscita
>Ad esempio se ho un tempo prefissato di 2sec, premo un tasto per un certo tempo e la bobina in uscita andrà subito a True, quando smetto di premere il tasto dopo 2 secondi il timer interrompe il flusso e porta la bobina in uscita a False
>

>[!note] **Timer TONR**
>Il Timer TONR accende una bobina solo se è stato dato un input per un certo tempo prefissato anche dilazionato in più parti
>Ad esempio ho un tempo prefissato di 2sec, mando un primo input in ingresso pari a 1sec, dopo qualche min mando un altro input in ingresso pari a 1sec la bobina in uscita andrà a True, essa resterà a True finché non verrà dato un input nella variabile reset ( R ) nel timer

>[!warning] **Attenzione**
>Tutti i timer possiedono una variabile ET, essa agisce come variabile di conteggio tenendo conto del tempo, la variabile associata a ET va scritta con la sintassi esempio %MD1 e il tipo di dato è time
