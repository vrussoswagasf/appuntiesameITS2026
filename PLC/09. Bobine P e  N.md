# Bobine P e N
---
## Componenti
| Nome  | Tipo di Dato | Indirizzo | Commento |
| ----- | ------------ | --------- | -------- |
| bt    | bool         | %I0.0     |          |
| stato | bool         | %M0.0     |          |
| Tag_2 | bool         | %M0.1     |          |
| led   | bool         | %Q0.0     |          |
## Descrizione
Le bobine P e N servono per rilevare il fronte di salita e discesa della bobina in uscita, in questo caso premo il pulsante bt, al rilascio viene rilevato il fronte di discesa di stato, nel secondo segmento stato va a True e fa partire un timer TP di 1sec accendendo un led per 1sec
