# Autoritenuta
---
## Componenti
| Nome | Tipo di Dato | Indirizzo | Commento |
| ---- | ------------ | --------- | -------- |
| BT1  | bool         | %I0.0     |          |
| BT2  | bool         | %I0.1     |          |
| LED  | bool         | %Q0.2     |          |
## Descrizione
L'autoritenuta serve per mantenere a True un Output portando solo una volta a True un Input
In questo caso se premo BT1 la bobina LED va True, poi il LED viene richiamato come Input e collegato tra BT1 e BT2, BT2 essendo normalmente chiuso chiude il collegamento e il LED resta acceso
Se premo BT2 il collegamento si interrompe e il LED si spegne
