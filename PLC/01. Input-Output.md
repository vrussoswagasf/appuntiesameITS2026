# INPUT OUTPUT
---
## Componenti
| Nome | Tipo di Dato | Indirizzo | Commento |
| ---- | ------------ | --------- | -------- |
| BT   | bool         | %I0.0     |          |
| LED  | bool         | %Q0.1     |          |
## Descrizione

Se premo il pulsante il contatto BT ( Input ) diventa True e la bobina LED ( Output ) va a True
