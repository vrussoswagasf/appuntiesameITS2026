# Marcia Arresto con Pulsante Singolo
---
## Componenti
| Nome  | Tipo di Dato | Indirizzo | Commento |
| ----- | ------------ | --------- | -------- |
| BT    | bool         | %I0.0     |          |
| LED   | bool         | %Q0.0     |          |
| stato | bool         | %M1.0     |          |
| Tag_1 | bool         | %M1.1     |          |
## Descrizione
Viene rilevato il fonte di salita di BT che accende una bobina in uscita stato.
Nel secondo segmento il LED all'inizio è spento, con i contatto normalmente chiuso fa passare il flusso fiché stato non va a True, quando arriva a True accende la bobina in uscita LED, successivamente il contatto LED essendo stato portato a Ture dopo l'accensione della medesima bobina in uscita incontra stato con contatto normalmente chiuso, se noi stiamo ancora premendo BT il contatto non farà passare il flusso, una volta rilasciato BT il flusso sarà continuo tra LED stato nc e la bobina LED, se premo una seconda volta BT il contatto stato normalmente chiuso si apre interrompendo il flusso e spegnendo la bobina LED
