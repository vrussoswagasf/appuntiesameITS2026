# Blink con Merker
---
## Componenti
| Nome      | Tipo di Dato | Indirizzo | Commento |
| --------- | ------------ | --------- | -------- |
| Clock_1Hz | bool         | %M100.5   |          |
| LEDVERDE  | bool         | %I0.0     |          |
## Descrizione
Nei PLC Siemens è possibile creare 8 Merker di clock associati a varie frequenze
Il Merker Clock_1Hz quindi diventerà True e False con una frequenza di 1Hz, di conseguenza accendendo e spegnendo il LEDVERDE
