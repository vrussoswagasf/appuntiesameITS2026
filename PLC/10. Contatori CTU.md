# Contatori CTU
---
## Componenti
| Nome      | Tipo di Dato | Indirizzo | Commento |
| --------- | ------------ | --------- | -------- |
| BT        | bool         | %I0.0     |          |
| LED       | bool         | %Q0.0     |          |
| conteggio | int          | %MW0      |          |
## Descrizione
Il contatore CTU serve oltre a contare quante volte ha ricevuto in input tramite CU con valore in uscita CV, quando raggiunge il numero di conteggi prefissato in PV, in questo caso 10, fa passare il flusso in uscita Q, in questo caso accendendo LED, successivamente c'è un blocco di comparazione il quale se rileva che il valore del conteggio è pari o superiore a 11 resetta il contatore CTU di conseguenza portando a zero conteggio e spegnendo il led, in breve per accendere il led devo premere BT per 10 volte, se la premo una 11esima LED si spegne, per riaccenderlo bisogna premere di nuovo per 10 volte BT
