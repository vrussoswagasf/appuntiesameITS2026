# Blink con Timer
---
## Componenti
| Nome      | Tipo di Dato | Indirizzo | Commento                                              |
| --------- | ------------ | --------- | ----------------------------------------------------- |
| FirstScan | bool         | %M1.0     | Merker di sistema che si attiva al primo scan del PLC |
| stato     | bool         | %M0.0     |                                                       |
| stato2    | bool         | %M0.1     |                                                       |
| Tag_1     | bool         | %M0.2     |                                                       |
| Tag_3     | bool         | %M0.4     |                                                       |
| LED       | bool         | %Q0.0     |                                                       |
## Descrizione
Alla prima scansione del PLC il timer TP1 accende la bobina stato per 1sec, di conseguenza accendendo la bobina in uscita LED nel secondo segmento sempre per 1 sec.
passato il tempo di 1sec il timer TP interromper il collegamento portando stato a 0, viene rilevato il fonte di discesa di stato e questo fa accendere il secondo timer TP2 che accende la bobina in uscita stato2 per 1sec.
Passato 1sec il timer TP2 interrompe il flusso e porta a 0 lo stato della bobina in uscita stato2, successivamente viene rilevato il fonte di discesa di stato2 che fa ripartire il timer TP1, risultato il led resterà acceso per 1sec poi si spegnerà per 1sec continuando poi il ciclo di accensione e spegnimento
